About

EKO is a simple sound editor.

Here is a nice screenshot:

To install EKO from the source, you need just a modern version of Qt 4, JACK sound server, libsndfile, libsamplerate and of course GCC.

EKO understands all popular sound formats (except of MP3) and useful in simple editing (cut/copy/paste) with a minimum FX processing. External fx currently are not supported.

Issues

If EKO crashes and JACK reports something like "Cannot lock down memory area (Cannot allocate memory)", try to follow these steps:

1. Under root, edit the /etc/security/limits.conf or /etc/security/limits.d/audio.conf file. Add there two lines:

@audio - rtprio 95
@audio - memlock unlimited

2. Add yourself (username) to the "audio" group: adduser username audio

3. Reboot.
